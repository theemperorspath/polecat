01 — detection
Minimum segment length
Ignore reflected values shorter than this. Lower catches short IDs but adds noise.
Match mode
Exact only, or also match when a source value is contained in a segment (e.g. 123 in 123.json).
Co-location guard
Suppress reflections that fall inside the shared path prefix of the page and request. Kills the "page fetching its own directory" false positive.
Dynamic values only
For path sources, only flag identifier-looking segments (has a digit, UUID, or long token). Query and hash sources are always checked.
Same-origin only
Ignore cross-origin requests entirely. CSPT is same-origin by nature, so this is usually safe and quieter.
02 — detectors
fetch / XMLHttpRequest
The core hook. Covers the vast majority of CSPT.
navigator.sendBeacon
Credentialed POST-style telemetry. Occasionally an overlooked sink.
EventSource (SSE)
Server-sent event stream URLs.
Resource loads
img / script / link / iframe / websocket via webRequest. Catches CSS-injection CSPT the page hook can't see.
03 — behavior
Badge counts
What the toolbar number tallies.
Clear on navigation
Reset records when the tab does a full page load. Turn off to accumulate across navigations.
04 — canary
Canary token
Seed this wherever your input persists, then browse. A hit means a stored/DOM value reached a request path.
05 — muted
Nothing muted. Mute noisy hosts or endpoints from a record in the popup.
